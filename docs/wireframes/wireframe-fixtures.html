Wireframe: Fixtures – Key Patterns
● Date strip: small dot below date = matches that day; active=blue pill
● Filter chips: scroll horizontally, multi-select for competitions
● Kickoff time: blue = today's matches, gray = future dates
● Notification bell: filled (active) = subscribed; tap to toggle goal/start alerts
● Odds row: 1X2 quick odds — 3 mono pills below team names (optional display)
● PPD badge: amber "PPD" tag replaces time + dims row opacity to 0.7
● Venue label: right-aligned below bell icon, 10px muted text
● Date groups: gray section headers, match count right-aligned
● Comp headers: flag + competition + round — 36px, no collapse needed
● Load more: lazy loads next day group on tap (not paginated)
Odds display is opt-in setting (default off for non-betting regions).
