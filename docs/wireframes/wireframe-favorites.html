Wireframe: Favorites – Key Patterns
● Live banner: pinned at top when saved teams are playing — auto-shows/hides
● Segment control: Teams / Competitions / Players — iOS-style pill switcher
● Team card: crest + name + position + league zone badge + filled heart
● Form dots: W/D/L circles, 20px, larger than standings (more space)
● Live indicator: pulsing dot + minute on form row if team currently playing
● Next match strip: opponent crest + name + competition + date/time
● Zone badge: colored dot + "CL/EL/ECL/REL" text next to heart
● Heart icon: filled red = saved; tap to remove (confirm dialog in real app)
● Add row: dashed border, blue on hover → opens team/comp search
● Notification CTA: deep link to per-team push notification settings
● Edit mode: long-press or "Edit" link → drag-to-reorder + swipe-to-delete
Empty state (not shown): centered heart icon + "Follow your first team" CTA button.
